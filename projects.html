A/B Testing Analysis

Category: Data Analysis

Date: January 2024

This project involved conducting an A/B test to evaluate the impact of introducing a banner on GloBox's mobile website, highlighting food and drink products. The analysis focused on understanding user conversion rates and average spending.

Steps Taken:

  1. Experiment Design: Developed the A/B test with control and treatment groups.
  2. Data Collection: Extracted user and purchase data from the GloBox database using SQL.
  3. Statistical Analysis: Performed using Python to compare conversion rates and average spending between the groups.
  4. Visualization: Created visual representations of the data using Tableau.
  5. Recommendation: Suggested further refinement of the banner design before a wide-scale launch based on the findings.

Key Findings:

  • Conversion Rates: Control Group: 4.2%, Treatment Group: 5.0%
  • Average Spending: Control Group: $86.0, Treatment Group: $73.2
  • The treatment group showed a higher conversion rate but lower average spending, indicating potential changes in user purchasing behavior due to the banner.

Further Details: For an in-depth analysis, refer to the Tableau Dashboard.
